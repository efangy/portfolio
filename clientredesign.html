Ceren Budak Website Redesign

For my Web Design, Development, & Accessibility Class, we were tasked to re-design professor Ceren Budak's website. We did this by first identifying problems heuristically with the original design and then creating a flow, wireframes, and prototypes for desktop, tablet, and mobile views. We then used CSS, HTML, and Javascript to code a fully responsive and accessible website.

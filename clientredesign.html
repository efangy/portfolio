Ceren Budak Website Redesign

For my Web Design, Development, & Accessibility Class, we were tasked to re-design professor Ceren Budak's website. I did this by first identifying problems heuristically with the original design -- heavily emphasising usability -- and then created a flow, wireframes, and prototypes for desktop, tablet, and mobile views. I then used CSS, HTML, and Javascript to code a fully responsive and accessible website with three separate media queries.
